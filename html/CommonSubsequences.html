Common Subsequences Algorithms Tutorial

Longest Common Subsequence Algorithm: Let x and y be the given strings. Fill in the cells in the table such that:

1: If i or j is equal to zero, then c[i, j] = 0.

2: If x[i] is equal to y[j], then c[i, j] = c[i - 1, j - 1] + 1.

3: If x[i] is not equal to y[j], then c[i, j] is equal to the bigger value between c[i, j - 1] and c[i - 1, j].


Shortest Common Supersequence Algorithm: Let x and y be the given strings. Fill in the cells in the table such that:

1: If i = 0, then c[i, j] = j.

2: If j = 0, then c[i, j] = i.

3: If x[i] is equal to y[j], then c[i, j] = c[i - 1, j - 1] + 1.

4: If x[i] is not equal to y[j], then c[i, j] is equal to the minimum value between c[i, j - 1] + 1 and c[i - 1, j] + 1.


Backtracking: For each cell in the table keep its predecessor cell and use it to trace back the final solution.
