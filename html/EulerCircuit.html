Euler Circuit Tutorial

Eulerâ€™s Theorem: A connected graph has an Euler circuit if and only if all the vertices in the graph have even degrees.

Algorithm description:

1: Choose a starting vertex X

2: Choose a vertex Y that shares an unused edge (X, Y) with vertex X.

3: Repeat step 2. for Y until there is no an unused edge incident to Y.

4: If no vertex in the obtained circuit C has an unused edge , terminate.

5: Otherwise, select a vertex X with an unused edge and follow steps 2. and 3. until you obtain a new circuit.

6: Merge the two circuits

7: Go to step 4.
