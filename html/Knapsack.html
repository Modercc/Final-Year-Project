Knapsack Problem Tutorial

Knapsack Problem: Let w and v be arrays such that w[i] and v[i] are numbers representing weight and value of the item i respectively and let w represent the current capacity of a knapsack. To obtain the solution of the 0/1 Knapsack problem fill in the table by following the rules displayed below:

1: If i = 0, then OPT[i, w] = 0.

2: If w[i] > w, then OPT[i, w] is equal to OPT[i-1, w].

3: If w[i] < w, then OPT[i, w] is equal to the bigger value

between OPT[i - 1, w] and v[i] + OPT[i - 1, w - w[i]].


Backtracking: For each cell in the table keep its predecessor cell and use it to trace back the final solution.
