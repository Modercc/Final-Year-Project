Dijkstra Algorithm Tutorial

Relaxation: The process of testing if we can find a shorter path to Y than the one obtained so far, by including a given edge (X, Y) in the constructed path.
The key is to maintain a set S of nodes to which the shortest path has already been found.

Dijkstra algorithm description:

1: Set S should include the starting vertex v.

2: Relax each vertex in the graph that is outside the set S using the last added vertex to the set S.

3: If there is at least one vertex outside of the set S, add to the set S the one with the minimal distance from the starting point found so far.


Backtracking: For each vertex keep its predecessor vertex and trace back the path to the starting vertex.
